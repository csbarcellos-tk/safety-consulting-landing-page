# 🏢 Landing Page - Massano Assessoria

Landing page institucional desenvolvida para uma empresa especializada em **segurança do trabalho e gestão de SST**.

---
# 🎯 Objetivo do Projeto

Desenvolver uma página profissional para apresentação de serviços de segurança do trabalho e geração de **leads qualificados**, com foco em conversão e presença digital da empresa.
